Mysql RDBMS -> sqoop 순으로 설치

[1] Mysql RDBMS 설치

1. mysql 다운로드 및 설치 : 설치시 pw는 root로 합시다
    sudo apt-get install mysql-server-5.7

2. mysql 실행
    mysal -u root -p

3. Database 생성 및 접속
    create database world;

    use world;

4. 다운받은 test data를 mysql rdbms에 구축하기 및 확인
    source world_innodb.sql;

    show tables;

5. 사용자 권한 추가 및 재접속 : 계정 및 pw = sqoop
    grant all privileges on world.* TO 'sqoop'@'localhost' identified by 'sqoop';

    flush privileges;

    quit;

mysql -u sqoop -p
