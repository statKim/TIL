# Today I Learned

## 작성 방법
- 프로그래밍 언어 이름으로 폴더 생성
- 각 언어별로 공부한 책 or 강의의 이름으로 폴더를 만들고 그 안에 코드 및 내용을 정리
- 추가로 글과 수식이 필요한 경우에는 코드 파일과 동일한 이름의 md 파일로 정리
- **etc** 폴더에는 프로그래밍 언어가 아닌 추가 내용이나 설명을 md 파일로 정리
